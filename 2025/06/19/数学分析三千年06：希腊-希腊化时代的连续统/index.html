2.1 希腊人塑造数学

当埃及和美索不达米亚文明繁盛之时，克里特岛上又发展出了一种更先进的文明。这种文明创立了米诺斯文化。公元前2千纪初，印欧部落移民到巴尔干半岛南部。在米诺斯文化的影响下，迈锡尼文明在大约公元前1600年至1000年间发展起来。

在公元前1200年左右，所谓的“海上民族”摧毁了地中海地区的大部分之后，多利安人入侵始于公元前1000年左右。多利安部落离开了他们位于希腊西北部（达尔马提亚）的故土，并扩散到伯罗奔尼撒半岛。其他希腊部落也融合在一起，最终形成了我们今天称之为“希腊人”的民族集合。

大约公元前750-500年的“黑暗时代”，整个地中海地区都被希腊人殖民，随后斯巴达在大约公元前550年建立了伯罗奔尼撒同盟。这开启了通往“经典希腊时期”的道路，而该时期直到亚历山大大帝时才转变为希腊化时代。

2.1.1 开端：米利都的泰勒斯及其门徒

在泰勒斯时代，小亚细亚的米利都是一个大型爱奥尼亚贸易城镇，但泰勒斯的出生年份尚不确定。根据第欧根尼·拉尔修可能迟至公元3世纪上半叶撰写的《著名哲学家传》[Diogenes Laertius 1931]，泰勒斯生活在公元前640年至562年之间；格里克[Gericke 2003]提到其生卒年为公元前624-548/545年。目前，其出生年份大约为公元前624年，去世年份大约为公元前546年似乎已获接受。可以确定的是，泰勒斯曾游历埃及，并将埃及的数学知识带到小亚细亚，从而引入了希腊人的文化领域。

我们没有泰勒斯的任何书面文献，但他的能力很早就受到赞扬，因此他被称为古代七贤之首。如果我们相信传统，那么泰勒斯一定是一个通才。据说他在公元前585年预测了一次日食，从而结束了米底人和吕底亚人之间一场长期战争。施拉姆已经澄清，预测月食的规律性并不需要卓越的天文学知识[Schramm 1994, p. 572f.]，因为在巴比伦楔形文字板中已经可以找到时间网格。然而，这对于日食而言并不成立，我们必须将这个故事归于神话范畴。我们遵循诺伊格鲍尔[Neugebauer 1969b, p. 142]：

“在特定地点可见的日食没有周期 [...] 公元前600年时，没有巴比伦理论可以预测日食 [...]”

图2.1.2. 米利都的泰勒斯以及米利都城门（近东博物馆，SMB [照片：H.-W. Alten]）

甚至更尖锐地指出 [Neugebauer 1975, p. 604]：

“因此，没有理由将‘泰勒斯日食’的故事视为巴比伦对最早希腊天文学产生影响的证据。所有现有资料都表明，直到三个世纪后才出现这种接触。”

最近，奥塔·文斯库斯（Otta Wenskus，生于1955年）在 [Wenskus 2016] 中再次审视了“泰勒斯日食”的案例，并清楚地表明我们所讨论的是一个传说而非真实故事。

据说泰勒斯也曾担任工程师，并以证明定理而闻名。后一种说法有些“爆炸性”！埃及人和美索不达米亚人的数学特点是严格的解决数学问题文化，即学生通过具体例子学习计算。我们没有证据表明在古风时代文化中存在一般数学定理的陈述（更不用说证明了）。只有具体问题被解决，甚至毕达哥拉斯定理，即在任何直角三角形中，斜边 c 的平方等于两条直角边 a 和 b 的平方和，也不是以公式 c² = a² + b² 的形式而闻名的，而只是以具体的（毕达哥拉斯）数对形式，如 5² = 4² + 3²。古希腊人感受到通过演绎进行推导的必要性，其重要性无论如何强调都不为过。因此，亚里士多德（公元前384-322年）将泰勒斯视为自然哲学推理方式的发明者 [Mansfeld 1999]。这尤其意味着泰勒斯将自然现象去神化，并使其能够进行理性解释。

泰勒斯标志着数学作为一门演绎科学的辉煌开端，并且数学成为希腊哲学的重要组成部分。甚至“数学”一词也源于希腊语，意为“属于学习之物”。因此，数学被视为古典教育的根本组成部分。

泰勒斯最著名的学生是阿那克西曼德（Anaximander）（生于公元前611年）和阿那克西米尼（Anaximenes）（生于公元前570年）。他们都扩展了其师父的自然哲学，但两人在分析学前史中肯定没有扮演任何角色。只有阿那克西米尼的学生阿那克萨哥拉（Anaxagoras）（公元前500-428年）为分析学史做出了贡献，而且他的工作对数学家们产生了强烈影响，直到19世纪！阿那克萨哥拉来自克拉佐门奈（Klazomenai），一个爱奥尼亚城镇，位于现代土耳其城市伊兹密尔以西约40公里处。他的宇宙学理论使他与同时代的人格格不入。他宣称太阳是一团炽热的铁，这导致他因不敬神而被起诉并入狱。第欧根尼·拉尔修 [Diogenes Laertius 1972, p. 143] 给出了关于审判结果的两种不同版本。由于伟大的政治家伯里克利（约公元前495-429年）是阿那克萨哥拉的学生，他得以使阿那克萨哥拉免受最坏的惩罚，阿那克萨哥拉被处以罚款并被流放。在另一个版本中，阿那克萨哥拉还被指控叛国并被判处死刑。你可以选择你更喜欢的版本。无论情况如何，阿那克萨哥拉似乎在狱中饱受无聊之苦，因此提出了以下任务：

圆的求积：
给定一个半径为 r 的圆。构造一个面积相同的正方形。

这里的“构造”一词蕴含着一些爆炸性的含义，因为在阿那克萨哥拉之后不久，不言而喻，“构造”意味着仅使用圆规和直尺进行构造。直尺不得有任何标记，必须被视为一种理想化的抽象工具。圆规也不得显示任何参考刻度，它也是一个抽象工具。此外，它被认为是可折叠的圆规；每次你把它从纸上移开时，它都会折叠，这样就不能用作分规。

由于可折叠圆规使用起来很麻烦（想象一下将给定长度转移到图纸的另一部分），欧几里得在公元前3世纪的著名著作《几何原本》中已经证明了“圆等价定理”作为第一卷命题2 [Euclid 1956, Vol. I, p. 244]，这最终允许使用普通圆规，而不会每次提起时都折叠。

圆的求积问题占据了几代数学家的心智，直到19世纪费迪南德·林德曼（Ferdinand Lindemann，1852-1939）证明数 π 是一个超越无理数，即它不是以下类型方程的根时，这个问题才得以解决：

anxn + an-1xn-1 + ... + a1x + a0 = 0

其中 a0, a1, ..., an 是有理系数。这最终证明了圆的求积根本不可能。

我们将回到阿那克萨哥拉，因为他在分析学史中扮演了另一个重要角色，但这里我们只提两件值得注意的事情：

（1）圆的求积问题最终被证明是关于无理数性质的问题；（2）无理数的存在问题似乎在希腊数学中扮演了重要角色。我们现在要阐明这一点。

2.1.2 毕达哥拉斯学派

在卢恰诺·德·克雷森佐（Luciano de Crescenzo，生于1928年）对希腊哲学史的概述中，他简单地称毕达哥拉斯（约公元前570年 – 约公元前496年）为“超级巨星”[De Crescenzo 1990, Band 1, p. 61 ff.]。毕达哥拉斯出生于公元前570年左右的爱奥尼亚岛萨摩斯，靠近米利都的沿海城镇。根据第欧根尼·拉尔修 [Diogenes Laertius 1931, S. 323] 的记载，毕达哥拉斯曾游历埃及和美索不达米亚，在那里他学习了数学；也许还有现在以他名字命名的著名定理——毕达哥拉斯定理。扬布里科斯（约公元250年-约325年）在他的《毕达哥拉斯生平》[Guthrie 1987, p. 57 ff.] 中写道，毕达哥拉斯被波斯国王冈比西斯（可能就是那位老国王）的士兵绑架到巴比伦。据说他在那里完成了算术、音乐和“所有其他科学”的学习 [Guthrie 1987, p. 61]。当毕达哥拉斯回到萨摩斯时，他发现这个岛屿处于僭主波利克拉特斯（Polycrates）的控制之下。尽管据说他曾担任波利克拉特斯儿子的老师，他最终还是流亡到意大利南部的克罗顿城（今卡拉布里亚的克罗托内）。据说他在那里建立了真正的贵族政体（=贤者统治）并立法。除了这些政治活动，他还在克罗顿建立了一所学校，并担任校长。然而，许多人认为他的思想具有煽动性，因此他不得不离开克罗顿。他在公元前510年左右在梅塔蓬图姆（今梅塔蓬托）找到了庇护所，并于公元前496年左右在那里去世。后来罗马人将梅塔蓬图姆的赫拉多利安神庙视为“毕达哥拉斯学派”。这个“学派”常被称为一个教派或秘密社团，因为它必须遵守奇怪的规定：

  • 禁食豆类，
  • 永不掰开面包，
  • 不吃心，
  • 不借火光修饰自己，
  • 起床后不搅动床铺；不要在上面留下身体的任何痕迹，

还有更多 [Guthrie 1987, p. 159 ff.]。这个团体一定非常有效：在很长一段时间里，毕达哥拉斯学派成员在希腊势力范围内的广大地区担任主要政治职务。我们更关心的是这个秘密社团的数学，它是从

大师传授给他的学生。我们从扬布里科斯那里知道，毕达哥拉斯学派至少有两种人：mathematikoi 和 akousmatikoi。前者是那些积极关注 mathemata — 学习课题 — 的人，而后者只是听他们所听到（akousmata）的人。今天我们会说前者是数学家，而后者是数学的爱好者。毕达哥拉斯学派的座右铭是：

“万物皆数！”

我们可以肯定，“数”指的是自然数，

N := {1, 2, 3, ...}.

要理解对自然数的热情，我们必须知道毕达哥拉斯留下了一个和谐的数学理论。

这个和谐理论可能起源于毕达哥拉斯演奏单弦琴，据第欧根尼·拉尔修记载，单弦琴也是他发明的。琴弦被拉伸在一个有十二个等距部分的刻度上。这种划分的选择似乎很巧妙，因为12可以被2、3、4和6整除，因此可以比较强烈缩短的琴弦发出的音调 [van der Waerden 1979, p. 370]。数字12，它的二分之一，三分之二和四分之三，

12, 9, 8, 6,

在毕达哥拉斯学派的思想中扮演着核心角色。它们构成了比例。

12 : 9 = 8:6;

其中9是12和6的算术平均数（9 = (12 + 6)/2），8是12和6的调和平均数（9 = 2/(1/12 + 1/6)）。当然，这种关系正是倾向于数论的毕达哥拉斯学派所期待的！如果同时出现的两个音调听起来令人愉悦，毕达哥拉斯学派称它们为symphon。八度音程、纯五度音程和四度音程就是这样的音高差异，它们是symphon。现在我们可以将2:1的关系分配给八度音程，3:2分配给五度音程，4:3分配给四度音程。从这些基本比例出发，毕达哥拉斯学派计算了许多其他比例，参见 [van der Waerden 1979, p. 367 f.]。

毕达哥拉斯学派还发现，一个音调可以以比例形式与另一个音调“比较”，这同样适用于长度、面积和体积。如果两个量 a 和 b 都是第三个量 c 的倍数，则称它们是可公度的（共同可测量的），即：

如果存在一个量 c 和两个数 m 和 n，使得 a = mc 且 b = nc 成立，则两个量 a 和 b 是可公度的。

这也可以用另一种方式表达，即比例：

如果 a : b = m : n 对于两个自然数 m 和 n 成立，则两个量 a 和 b 是可公度的。

我们必须注意到，在希腊人看来，可比较的量总是必须是同种的，即等比例或维度相同的。不允许将面积与长度或体积与面积进行比较，而只能将长度与长度等进行比较。

毕达哥拉斯学派通过比例得到了正有理数，即用现代话说，他们得到了正分数集 Q+。1对他们来说，没有其他数存在——它们被禁止存在：自然数（“万物皆数”）和比例构成了现实的本质。它们是存在的典范。如果“万物皆数”，那么一切都是可公度的，因此可以在同一尺度上测量。在他们的世界里，没有什么是不公度的！然而，毕达哥拉斯学派的这个基本信念被他们自己的一位成员打破了！瓦德登（van der Waerden）将梅塔蓬图姆的希帕苏斯（Hippasus of Metapontum）的生卒年定为公元前520年至480年2 [van der Waerden 1979, p. 74]，据说他展示了不公度量的存在，从而引发了数学史上第一次根本性危机！人们经常读到，他的存在性证明是在秘密符号下进行的

希帕苏斯是否真的选择了毕达哥拉斯学派的神圣符号，或者他根本没有证明任何东西，我们不得而知。至少，根据毕达哥拉斯定理，单位正方形的对角线长度为√2，在这种更简单的情况下，无限倒数减法也是证明1和√2不可公度性的首选方法。

数学史上的一个传说认为，不可公度量的发现震撼了毕达哥拉斯学派的核心。扬布里科斯告诉我们 [Guthrie 1987, p. 116]：

“因此，据说第一个向不配接受它的人泄露可公度量和不可公度量理论的人，受到了毕达哥拉斯学派的痛恨，他们不仅将他逐出他们的共同社团，并禁止他与他们共同生活，而且还为他建造了一个[象征性]坟墓，如同一个已经迁移到另一世的人。还据说神圣的力量对那个泄露了在球体中内接正十二面体方法的人非常愤怒，正十二面体是所谓的立体图形之一，构成五角星形。但据其他人说，这才是那个揭示了无理量和不可公度量学说的人的遭遇。”

十二面体是一个由12个五边形构成的立体图形，如果我们阅读扬布里科斯 [Guthrie 1987, p. 79]，这里就和希帕苏斯联系起来了：

“至于希帕苏斯，他们承认他是毕达哥拉斯学派的一员，但由于他泄露并解释了从十二个五边形形成球体的方法，他遭遇了不敬神的命运，葬身大海；[...]”

这就是完整的民间传说：梅塔蓬图姆的希帕苏斯发现了不可公度数的存在。这是一件极其可憎的亵渎行为，以至于他自己的毕达哥拉斯朋友设计了一场阴险的暗杀，他们将载有希帕苏斯的船只凿沉，导致希帕苏斯溺水身亡。

我们有充分的理由不信任这种民间传统。列奥尼德·兹穆德 [Zhmud 1997, p. 173] 认为，柏拉图（公元前428/427-348/347年）在他的对话《大希庇亚斯》中使用的“arrhetos”一词，在指代无理数时，确实翻译为“秘密”或“不可言喻的谜”，但它的意思不过是“不能用数字表达”。兹穆德解释说，关于梅塔蓬图姆的希帕苏斯的报道是由于早期评论家的翻译错误。唯一得到证实的是希帕苏斯与毕达哥拉斯学派之间出于政治动机的争吵。无理数的发现也可能根本没有违反任何不成文的毕达哥拉斯定律！另一位现代批评家是大卫·福勒（David Fowler，1937 – 2004）[Fowler 1999, p. 289 ff.]，他详细研究了基础危机民间传说的历史和分析。尽管福勒的著作 [Fowler 1999] 也不能给出明确的答案，但它提供了一些有趣且发人深省的想法，这些想法以前并非数学史家的关注焦点。

关于无理数发现所引发的根本性危机，其民间传说从何而来？在20世纪初数学基础领域真正动摇的危机达到顶峰时，数学家赫尔穆特·哈塞（Helmut Hasse，1898 – 1979）和逻辑学家兼哲学家海因里希·肖尔茨（Heinrich Scholz，1884 – 1956）共同撰写了一篇关于希腊数学基础危机的论文 [Hasse/Scholz 1928]。这篇论文很可能引发了至今仍在流传的希腊数学危机故事。

无论如何，某些长度不可公度的发现似乎已经使毕达哥拉斯学派的算术几何化计划停滞不前。这种停滞也可能导致希腊数学中几何学的良好发展，而算术和代数却受到草率对待。至少，伟大的哲学家柏拉图在他的《泰阿泰德篇》147d-148a和《法律篇》819d-822d中宣称，他认为希腊人没有着手研究不可公度量问题是一种耻辱，参见 [Popper 2006, p. 329]。

2.1.3 欧几里得《几何原本》中的欧多克索斯比例论

欧几里得的《几何原本》[Euclid 1956] 对希腊数学进行了精彩的总结。这部著名的著作分为13卷，归功于亚历山大里亚的欧几里得（约公元前300年），尽管我们对他几乎一无所知。第十卷包含对可公度量和不可公度量的公理化处理，并解释了（无限）倒数减法的方法。在第十卷的结尾，我们发现了单位正方形中对角线与其边不可公度的证明。如果我们追随库尔特·冯·弗里茨 [von Fritz 1971, p. 562]，这个在正方形上进行的证明是原始的。

然而，更有趣的是第五卷，其中包含克尼多斯的欧多克索斯（Eudoxus）的比例理论。欧多克索斯被认为是柏拉图学院中最具天赋的数学家，他与柏拉图学院保持着密切联系。现代分析的基础是我们现在称之为阿基米德公理的：

公理：对于任何非常小的正实数 ε，存在一个自然数 n，使得：

0 < 1/n < ε

成立。

这个公理，如今常被认为是上确界原理的推论，对实数的结构提供了决定性的洞察。例如，根据这个公理，不存在最小的正实数。因为如果 δ > 0 是这个最小的数，那么就可以找到一个自然数 n，使得 1/n 介于 0 和 δ 之间。因此 1/n 将比 δ 更小。这个公理是希腊人对无限小问题真正的答案：不存在无限小的数！因此，所有问题都应该已经解决，但无限小的概念太有趣了，不能就这样放弃。

我们可以用以下方式等效地重述阿基米德公理：对于任意两个量 y > x > 0，都存在一个自然数 N，使得 N ⋅ x > y 成立。正是这种形式的阿基米德公理进入了欧几里得的《几何原本》。这个公理本应被称为欧多克索斯公理。在第五卷的开头，我们找到了定义4 [Euclid 1956, Vol. II, p. 114]：

“4. 量被称为彼此有比的，当它们相乘时，能够超过彼此。”

并继续写道：

“5. 当第一和第三的任意等倍数，以及第二和第四的任意等倍数，前者的等倍数同样超过、同样等于、或同样不足后者的等倍数时，量被称为处于同一比例中，即第一与第二，以及第三与第四。”

今天我们会将此表述为：

定义5 欧多克索斯/欧几里得：我们定义 a : b = c : d 当且仅当对于所有自然数 m, n 都满足：

  • 如果 n ⋅ a > m ⋅ b 则 n ⋅ c > m ⋅ d
  • 如果 n ⋅ a = m ⋅ b 则 n ⋅ c = m ⋅ d
  • 如果 n ⋅ a < m ⋅ b 则 n ⋅ c < m ⋅ d.

现在看来，欧多克索斯似乎只是模糊了比例或比率看似清晰的定义，并留下了一个难以理解的定义。但事实远非如此！这个定义是一种特殊的突破。如果出现两个不可公度量 a 和 b，会发生什么？那么欧多克索斯的定义将有理数 m/n 分为两个不相交的集合；即一个集合 U，其中满足上述定义5中的第一种可能性。集合 U 的成员是那些 m/n，对于它们来说，

n ⋅ a > m ⋅ b, 因此 m/n < a/b

成立。由于假定 a 和 b 是不可公度的，因此定义5中的中间可能性被排除，但不是最后一种可能性。这里现在定义了一个集合 O，包含所有满足以下条件的 m/n：

n ⋅ a < m ⋅ b, 因此 m/n > a/b

因此我们已经确立了对有理数的划分：

Q = U ∪ O

将有理数划分为一个下集 U 和一个上集 O，两者不相交：

U ∩ O = ∅.

集合 U 中的每个数都小于集合 O 中的任何数。在 U 和 O 之间的“界面”处，可以定义一个新数，它显然必须是一个无理数。我们不得不等到19世纪下半叶，欧多克索斯的比例理论才被用于实数的构造。这个根本性的步骤最终由德国布伦瑞克的数学家理查德·戴德金（Richard Dedekind，1831-1916）完成。他“戴德金分割”的基础正是通过欧多克索斯对比例的定义实现的有理数不相交分割，如上所述！我们将在第9.7.1节回到戴德金和他的分割。

比例定义的另一个进步是现在可以比较“不同种类”的量。现在 a 和 b 可以是长度，而 c 和 d 可以是体积；它们的比率现在可以比较了。

但是欧多克索斯用阿基米德/欧多克索斯公理来解决什么任务呢？这在欧几里得《几何原本》的第五卷中也很清楚，当

a:c = b:c => a=b

被证明时（命题9, [Euclid 1956, Vol. II, p. 153 f.]）。我们遵循 [Edwards 1979, p. 14]：假设 a > b。根据阿基米德公理，存在一个自然数 N，使得

N ⋅ (a - b) > c

成立。此外，存在一个最小的自然数 M 具有属性 M ⋅ c > N ⋅ b，但它也满足 N ⋅ b > (M - 1) ⋅ c，因为 M 是满足 M ⋅ c > N ⋅ b 的最小自然数。将不等式 N(a - b) > c 和 Nb > (M - 1)c 相加得到

Na > Mc,

但根据我们的假设，Nb < Mc！这与比例的定义相矛盾。因此，我们假设 a > b 必须是错误的。

图2.1.10. 欧几里得（牛津大学自然历史博物馆的雕像）[照片：Thomas Sonar]

这种证明方式称为“反证法”（reductio ad absurdum），因为一个假设导致了矛盾。现在只剩下两种可能性：要么是 a < b，要么是 a = b。根据排中律（principle of tertium non datur），我们只需证伪 a < b，这与上述证明类似。总而言之，这个证明是一种双重反证法。我们很快会回到这种技术。

2.1.4 穷竭法——希腊式的积分

欧多克索斯利用阿基米德公理也开创了一种计算面积的方法：穷竭法。穷竭法这个名称由格雷戈里·德·圣-文森特（Grégoire de Saint-Vincent）（参阅第7.2节）于1647年提出 [Jahnke 2003a, p. 18]。曲线边界图形的面积通过多边形填充，并研究了一系列越来越好地穷竭该图形的多边形。

这种方法的基础可以在欧几里得《几何原本》第十卷命题1中找到 [Euclid 1956, Vol. III, p. 14]：

“给出两个不等量，如果从较大的量中减去一个大于其一半的量，再从剩余的量中减去一个大于其一半的量，如果这个过程持续重复，最终会留下一个量，它将小于最初给出的较小量。”

仔细阅读，我们发现这是对阿基米德公理的明显重述。用现代术语来说，我们从两个正量 G0 和 ε 开始，然后计算中间量：

G1 < (1/2)G0, G2 < (1/2)G1, G3 < (1/2)G2, 等。

直到我们达到某个指数 n，使得 Gn < ε。证明这种策略能成功遵循多步考量：

在步骤1中，阿基米德公理保证了可以选择一个自然数 n，使得 (n + 1)ε > G0 成立。由于 ε 是正数，因此 nε > ε，并且 nε + ε ≥ 2ε，因此

ε ≤ (n+1)ε / 2

因此我们有

nε > G0 - ε ≥ G0 - (n+1)ε / 2

除以2并合并相关项得到

(3/4)nε + ε/4 ≥ G0/2

但现在 nε ≥ ε，所以

nε ≥ (3/4)nε + ε/4 ≥ G0/2

即 nε > G0/2 > G1，我们已经完成了第一步。

在第二步中，我们从 nε > G1 开始。由于 (n - 1)ε ≥ ε 成立，因此 nε ≥ 2ε，或

ε ≤ nε / 2

因此我们得到

(n - 1)ε > G1 - ε ≥ G1 - nε / 2 .

除以2并合并项得到

(3/4)(n-1)ε + ε/2 ≥ G1/2 ,

并且由于 (n - 1)ε ≥ ε，因此

(n - 1)ε ≥ G1/2 > G2,

这样第二步也完成了。以这种方式继续，最终导致

ε > Gn

在步骤 n 中。

现在我们可以将这个版本的阿基米德公理应用于通过穷竭法计算面积。

定理（穷竭法在计算圆面积中的应用）：给定一个面积为 A(C) 的圆 C 和一个数 ε > 0。那么存在一个内接正多边形 P，使得

A(C) – A(P) < ε.

换句话说，增加内接于圆的多边形的边数最终会使该多边形的面积与圆的面积的差异不超过任意给定的正小量 ε。穷竭法现在允许在不依赖严格极限定义的情况下证明该定理。为此，我们从如图2.1.11(a)所示的内接正方形 P0 = EFGH 开始。为了引用阿基米德公理，我们写道

作为穷竭法的最后一个例子，让我们考虑欧几里得《几何原本》第十二卷命题2中的以下定理 [Euclid 1956, Vol. 3, p. 371]：

定理：如果 C1 和 C2 是半径分别为 r1 和 r2 的圆，那么以下关系成立：

A(C1) / A(C2) = r1² / r2²

（“圆的面积之比等于其直径的平方之比。”）

这个定理的证明是通过双重反证法完成的。因为只有三种可能性：

A(C1)/A(C2) = r1²/r2², 或 A(C1)/A(C2) < r1²/r2², 或 A(C1)/A(C2) > r1²/r2².

我们提出假设1：A(C1)/A(C2) < r1²/r2²，因此我们假设

A(C2) > A(C1)r2² / r1² := S.

那么数 ε := A(C2) – S 将是正数，即 ε > 0。根据上述圆面积穷竭法定理，在圆 C2 中存在一个内接多边形 P，使得 A(C2) – A(P) < ε = A(C2) – S 成立，即 A(P) > S。现在我们在 C1 中内接一个与 P 对应的多边形 Q。我们现在有（参见图2.1.12）

A(Q) / A(P) = r1² / r2²

图2.1.12. 圆中正多边形

因此

A(Q) / A(P) = r1² / r2² = A(C1) / (A(C1)r2² / r1²) = A(C1) / S

但由此可知

S / A(P) = A(C1) / A(Q) > 1,

因此 S > A(P)，这与我们假设 A(P) > S 相矛盾。于是，假设1必定是错误的，我们已经用反证法证明了这一点。

现在我们陈述假设2：A(C1)/A(C2) > r1²/r2²，并像上面一样证明其导致矛盾。我们现在已经完成了双重反证法，只有 A(C1)/A(C2) = r1²/r2² 才是正确的可能性。

2.1.5 角的难题

自欧多克索斯时代以来，希腊数学一直受阿基米德数系的支配，即阿基米德公理成立的数系：对于任何两个正数 x < y，总能找到一个自然数 n，使得 n \sdot x > y 成立。但欧多克索斯早就知道，其他数系——即所谓的非阿基米德数系——也是可以想象的 [Becker 1998, p. 104]。希腊人已经知道的一种非阿基米德量系是角或“角间角” [Thiele 2003, p. 1f.]。这些是两个相切圆之间或圆与其切线之间的角度，如图2.1.13所示。在欧几里得《几何原本》的第三卷中，我们找到了关于角的命题16 [Euclid 1956, Vol. II, p. 37]：

“自圆的端点垂直于直径所画的直线将落在圆外，并且在直线和圆周之间的空间中，不能再插入其他直线；此外，半圆的角度大于任何锐角直线角度，而剩余角度小于任何锐角直线角度。”

在圆周和切线之间，根本无法再画出落在圆外的直线。角的概念构成了一个非阿基米德数系，因为在任意两个角的案例中，阿基米德公理显然不成立。如果我们把角定义为切线之间的角度，那么每个角都简单地为零，阿基米德数系又恢复了。正如克尔（Körle）所写 [Körle 2009, p. 29]：

图2.1.13. 角或“角间角”

“对我们来说，根本不存在角。它们的问题是心理性质的。人们不知道如何抵御这样一种观念，即必须有什么东西来填补这个空白。它们无法被排除，但随着极限概念的出现而失效。无论如何解释，角量都只剩下零。关于角的争议持续了很长时间，甚至莱布尼茨也关注过它们。”

（对我们而言，根本不存在角。它们的问题是心理学性质的。人们无法抗拒某种东西必须填补这个空白的观念。它们无法被排除，而是随着极限概念的出现而变得没有意义。无论如何解释，这些角量最终都只会是零。关于它们的争议持续了很长时间，甚至莱布尼茨也曾关注过它们。）

我们稍后将看到，在非阿基米德数系中分析也是可能的。在这样的数系中，不同的角可以有不同的大小！

2.1.6 古代的三大经典问题

我们已经报道了圆的求积问题及其作者是狱中的阿那克萨哥拉。我们还必须提及另外两个问题，它们与圆的求积一起，在分析学史上扮演了重要角色。所有这三个问题都被称为“数学的经典问题”；参见 [Alten et al. 2005] 和 [Scriba/Schreiber 2000]。

三等分角：
给定一个角 α。仅用直尺和圆规精确地三等分这个角。

希思 [Heath 1981, p. 235] 曾猜测，这个问题起源于一个时期，那时人们已经能够用直尺构造五边形。

并希望构造更多多边形。实际上，构造一个正十边形需要将一个角三等分。由于圆的求积是一个不可解的问题，三等分角也是不可解的，正如现代代数所揭示的那样。几代数学家曾试图解决这个问题。

立方倍积：
给定一个体积为 V 的立方体。仅用直尺和圆规构造一个体积为两倍的立方体。

传说爱琴海基克拉泽斯群岛提洛岛的居民德洛伊人受到了瘟疫的困扰。他们向神谕求助，神谕建议将德洛伊神庙中立方体形状的祭坛倍增。由于未能成功，德洛伊人向伟大的哲学家柏拉图求助，柏拉图回答说，神灵实际上不需要一个新的祭坛，而是提出了这个问题，以羞辱德洛伊人，因为他们对数学毫无兴趣并鄙视几何学，参见 [Heath 1981, p. 245f.]。

关于圆的求积

对圆进行求积的尝试似乎对希腊数学家来说是一种诱惑。基奥斯的希波克拉底（公元前5世纪中叶或后半叶）发展了一种不基于穷竭法的方法，该方法甚至对近代学校使用的数学书籍都产生了影响。该方法依赖于“希波克拉底月牙形”。希波克拉底因此遵循了计算由部分圆周界定的面积这一不那么雄心勃勃的任务。

给定一个边长为1的正方形OACB和一个内切的四分之一圆BRA。我们设想边BC以恒定速度向下移动至OA。同时，边OB以恒定速度绕点O沿径向方向旋转至OA。假设两条边同时到达OA。在任意时刻，BC到达DE，OB位于OR位置。交点Q被定义为割圆曲线上的一个点。

现在我们追踪点R的运动。它的坐标由以下公式描述：

x = cos Θ y = sin Θ,

其中角度Θ从Θ = π/2 (=90°) 变为Θ = 0，如图2.1.16(b)所示。边BC以恒定速度沿y方向移动。我们用vy表示这个速度，并将BC的移动与角度Θ的移动联系起来，得到

y = vy · Θ.

如果Θ = 0，那么y = 0。如果Θ = π/2，那么y = 1。这些条件导致以下方程：

y = 1 = vy · π/2,

因此BC的速度必须是vy = 2/π。因此，角度与y坐标之间的关系是

Θ = πy / 2

但是

y / x = sinΘ / cosΘ = tanΘ,

因此tan(πy/2) = y/x，从而得到

x = y · cot (πy/2)

这就是割圆曲线的方程，当然希皮亚斯并不知道。由于方程中出现了余切，它是一个超越函数。三等分角可以通过以下方式完成。对于一个角Θ，对应一个特定的y值，参见图2.1.16(b)，由于Θ和y通过Θ = πy/2连接，我们只需要将高度为y/3的水平线与割圆曲线相交（交点为T），即可三等分角Θ。割圆曲线在圆的化方中的使用要复杂得多，参见[Heath 1981, Vol.1, p. 227f.]。

关于立方体倍增问题

给定一个边长为a的立方体，其体积为V = a³。如果要构造一个体积为原立方体两倍的新立方体，那么新边长x必须满足x³ = 2 · a³，即

x = ³√2 · a.

现代代数（仅在19世纪发展起来）告诉我们，这个数x不能用直尺和圆规构造，因为³√2不是一个可构造数。第一个严谨地证明立方体倍增和角的三等分问题不可解的荣誉归功于法国数学家皮埃尔·洛朗·旺策尔（Pierre Laurent Wantzel，1814-1848），他于1837年在利奥维尔的《纯粹与应用数学杂志》上发表了他的证明[Cajori 1918]。

我们不想描述希腊人解决这个问题的众多巧妙尝试，但希俄斯的希波克拉底因其突破性的发现而获得了永恒的荣誉。他成功地将立方体倍增问题简化为确定两个比例中项的问题。边长为2a的立方体显然不符合我们的条件，因为它只是我们开始的立方体的边长的两倍，但其体积却是8a³。然而，边长倍增的想法不知何故一直困扰着希波克拉底。他寻找介于a和2a之间的两个数，这两个数被称为两个比例中项。这里，如果a : x = x : b，那么x是a和b两个数的比例中项。

解这个比例式可得x² = a·b 或 x = √(ab)，因此比例中项不过是a和2a的几何平均值，其定义为

a : x = x : 2a,

因此x = √2 · a。通过仅仅一个比例中项，无法解决我们的立方体倍增问题，即x = ³√2 · a。这种认识很可能源于毕达哥拉斯学派，因为柏拉图在他的对话《蒂迈欧篇》中写道[Plato 1929, p. 59, 32A-B]：

“然而，仅仅两样事物不可能结合而没有第三者；因为它们之间必须有某种中间纽带将两者连接起来。而最美好的纽带，是它本身最完美地将自身与它所连接的事物合为一体；以最美好的方式实现这一点，是比例的自然属性。因为，每当三个数（无论立方数还是平方数）的中间项满足‘第一项与中间项之比，如同中间项与最后一项之比’时，反之亦然，‘最后一项与中间项之比，如同中间项与第一项之比’时，那么中间项就依次成为第一项和最后一项，而第一项和最后一项也依次成为中间项，其必然结果是所有项都可以互换，并且互换后它们形成一个统一体。现在，如果宇宙的形体本应是平面，没有深度，那么一个中间项就足以将自身和与其关联的各项连接起来；但现在情况并非如此：因为它必须是立体的，而使立体合一的绝不是单独一个中间项，而是始终需要两个。”

希波克拉底也关注一个实体，即立方体。因此，一个比例中项不足够。所以他寻找a和2a的两个比例中项x和y，使得

a : x = x : y = y : 2a

成立。从这些比例关系中可以得出三个方程，即

x² = a · y, y² = 2a · x, x · y = 2a².

将最后一个方程解出y，并将其结果代入第一个方程，得到立方体倍增的方程：

x³ = 2 · a³ ⇔ x = ³√2 · a.

显然，希波克拉底并未成功更接近立方体倍增问题的解决。但认识到立方体倍增问题完全等同于找到两条线段的两个比例中项，这只能被称为天才的发现！

当然，后来的希腊数学家试图解决立方体倍增问题，并在其数学方法的发展方面取得了令人瞩目的进步。我们必须提到狄奥克勒斯（Diocles，约公元前240年-约公元前180年）以及以他命名的藤蔓线（或常春藤曲线），通过它可以在几何上构造两个比例中项。在此背景下，我们还必须提及塔伦图姆的阿基塔斯（Archytas of Tarentum，公元前428年-公元前347年），他提出了一种卓越的三维构造方法，以确定所求新立方体的边长x。他成功地相交了至少三个旋转体，其中独特的交点就是所求的x。

在希波克拉底的两个比例中项方法处理立方体倍增问题的过程中，梅内克姆斯（Menaechmus，公元前380年-公元前320年）发现了圆锥曲线。然而，圆锥曲线得名并被深入分析是在后来由阿波罗尼奥斯完成的。

评论

希腊数学家的洞察力和创造力即使从现代角度来看也令人印象深刻。尽管他们能够通过某些曲线（如蚌线、圆锥曲线和割圆曲线）以任意精确度解决圆的化方、立方体倍增和角的三等分问题，但实际任务——仅用直尺和圆规进行精确构造——被证明是完全无法解决的。然而，即使在今天，在分析、代数和数论发展之后，仍有一些业余数学家相信他们已经成功解决了古代三大问题中的至少一个，或者证明了π的有理性。这方面精彩的例子可以在安德伍德·达德利的著作[Dudley 1987]中找到。任何试图阻止这些伪数学家继续游戏的行为都注定失败；他们要么根本不理解问题定义，要么缺乏必要的概念和数学知识。他们通常发明近似方法，这些方法在有限步骤内就能产生惊人好结果，但他们拒绝承认精确解只能在无限步骤之后才能得到（因此不可能成为希腊人所寻求的解决方案）。

2.2 连续体与原子 – 无穷小与不可分

不可公度量的发现，即无理数的存在，可能已经扰乱了希腊数学家，甚至可能是希腊人转向几何学的原因。无理数同时也是不可言喻的、不可理解的、不可形象化的[Lasswitz 1984, Vol. 1, p. 175]。然而，一场关于“存在”的哲学争论几乎使分析崩溃，这种冲击甚至在今天仍能感受到。我们不想深入探讨哲学问题，但建议读者参考相关文献，例如[von Fritz 1971]。然而，为了更好地理解数学和历史背景，肯定有必要提几句话。

2.2.1 埃利亚学派

由于与波斯人在爱奥尼亚海岸的战事，一些希腊人于公元前545年迁往南意大利西海岸。他们在那里建立了埃利亚定居点，即今天的维利亚。在埃利亚，一个名为埃利亚学派的哲学家群体逐渐形成。诗人兼自然哲学家色诺芬尼（Xenophanes，约公元前570年-约公元前475年）被视为其创始人。

图2.2.1. 巴门尼德；埃利亚的芝诺 [图片来源：Sailko]

真正伟大的埃利亚学派人物之一是巴门尼德（Parmenides，约公元前540/535年-约公元前483/475年），他为希腊哲学引入了新的思维方式。在巴门尼德之前的哲学家热衷于理解世界时，巴门尼德现在提出了非经验理论绝对确定性的主张，这些理论不能直接用于描述世界[Parmenides 2016, p. 4ff.]。“存在”（‘being’）成为巴门尼德哲学的核心，而存在（或逻各斯、合一、或神 [De Crescenzo 1990, Vol. 1, p. 112]）是独一无二的、整体的、不可动的。没有虚空，也没有“生成”；“非存在”是不可思议的。既然存在是不可动的，巴门尼德显然怀疑运动的可能性——我们只看到人类表面的运动，而实际存在是静态的——这一点被他最著名的学生，埃利亚的芝诺（Zeno of Elea，约公元前490年-约公元前430年） splendidly承认。柏拉图在他的对话《巴门尼德篇》[Plato 1939, p. 205]中记载，芝诺在128d中试图捍卫他的老师，反对“如果运动被否定则导致荒谬后果”的指责。但这一切在数学上意味着什么呢？

2.2.2 原子论与连续体理论

我们所知的关于芝诺的一切，几乎都来自于伟大的哲学家亚里士多德的著作，他塑造了西方世界几个世纪以来的思想。在从泰勒斯到苏格拉底时代的一些哲学家中，没有留下任何书面著作。这些哲学家被称为前苏格拉底派。我们所拥有的唯一材料以碎片形式流传下来[Early Greek Philosophy 2016]，这些碎片仅由后来的哲学家记录。

图2.2.2. 阿布德拉的德谟克利特；钞票细节（1967年希腊100德拉克马）

在亚里士多德的《物理学》中，这位伟大的哲学家用整本书——第六卷——来讨论连续体的问题[Aristotle 1995, p. 390-407]。正是在那里，芝诺有机会发言。

关于时间流逝和空间结构的两种思潮在希腊思想家中很流行：原子论和连续体理论。哲学家留基伯（Leucippus，公元前5世纪）和他的学生德谟克利特（Democritus，约公元前460年-约公元前370年）被认为是原子论的发明者。根据他们的思想，万物都由无限小的量——“原子”（atomon = 不可分）组成。然而，我们绝不能混淆我们现代对原子的理解与德谟克利特所说的原子。德谟克利特的“原子”在其原始意义上可能仍可进一步分割，但就我们讨论其数学含义而言，我们应该将原子想象为直线上的一点。这个点就是原子，根据德谟克利特的理论，整条直线是由无限多个点组成的。亚里士多德和许多其他人，其中一部分可以归因于芝诺，他们绝大多数都反对这种原子理论，我们将在下文看到。根据连续体理论，一条直线是一个“连续体”，它可以被任意分割。即使一个连续体被任意多次分割，总会留下一个仍然可以继续分割的连续体。分割过程永远不会导致一个点！因此，点不能是直线的元素！

在《物理学》第五卷中，亚里士多德引入了“在一起”、“分开”、“接触”、“相继”、“连续”等概念[Aristotle 1995, p. 383]

“当事物处于同一主要位置时，就说它们在空间上‘在一起’；当它们处于不同位置时，就说它们‘分开’。当事物的末端彼此相接时，就说它们‘接触’。”

他接着说 [Aristotle 1995, p. 383]：

“当事物在位置、形式或某些其他方面被明确地视为在开始之后时，它就是‘相继’的，并且在它与所相继的事物之间，没有同类的事物，例如：如果是线，则没有线或多条线；如果是单位，则没有单位或多个单位；如果是房屋，则没有房屋（不排除中间有不同类型的事物）。[……]相继且接触的事物是‘连接’的。‘连续’是连接的细分：当每个事物的接触边界变成一个共同的、同一个边界，并且如其名称所示，相互包含时，就称其为‘连续’：如果这些末端是两个，则不可能有连续性。”

这一定义在第六卷中被用来对原子论思想施以致命一击[Aristotle 1995, p. 390f.]：

“现在，如果‘连续’、‘接触’和‘相继’这些术语被理解为如上所述的定义——事物如果末端合一则连续，如果末端相接则接触，如果中间没有同类事物则相继——那么任何连续的事物都不能由不可分的事物组成：例如，一条线不能由点组成，因为线是连续的而点是不可分的。”

欧几里得在其《几何原本》的开头定义[Euclid 1956, Vol. I, p. 153]：

  1. 点是无部分者。
  2. 线是无宽度的长度。

他巧妙地避免了任何微妙的讨论。将“无宽度的长度”作为线的转述，简单地排除了任何关于原子或连续体的批评。但如果一个点没有部分，那么亚里士多德问道，一条线如何由点构成？线上的两个点又应在何种意义上相邻？

这些问题一直困扰着思想家，直到今天。我们提醒读者数学家赫尔曼·外尔（Hermann Weyl，1885-1955）早在1917年就撰写了关于连续体的文章[Weyl 1917]（英文译本参见[Weyl 1994]），并在1946年（他晚年）讨论了数学的哲学问题。在[Weyl 2009, p. 41]中，他从现代角度看待连续体，并与现代分析建立了桥梁，他写道：

“个别的自然数构成了数论的主体，自然数的所有可能集合（或无限序列）构成了连续体理论的主体。”

他还引用了阿那克萨哥拉的话来描述连续体的性质[Weyl 2009, p. 41]：

“在小的之中没有最小的，但总有更小的。因为存在的事物无论如何细分都不能停止存在。”

这个引用指的是阿那克萨哥拉的所谓《残篇3》[Schofield 1980, p. 80]：

“因为在小的之中没有最小的，但总有更小的（因为存在的事物不可能不存在）”

这被认为与芝诺悖论有关，我们将在下文讨论。

2.2.3 不可分体与无穷小量

原子论者和连续体支持者之间冲突的另一个原因是涉及到无穷大，参见[Heuser 2008, p. 59ff.]。德谟克利特和原子论者直言不讳地陈述了实际无穷大的存在，因为一条直线（或仅仅一个线段）包含着实际无穷多的点，即原子。亚里士多德和许多其他人在芝诺生前就已经拒绝实际无穷大的存在，并提出了“潜在无穷大”的观点，即分割过程可以一直持续下去。这场争论，对我们来说似乎多么“古老”，至今仍未停止！直到格奥尔格·康托尔（Georg Cantor，1845-1918）通过集合论的创立，才将实际无穷大严谨地引入数学。在康托尔的数学中，一条线（例如实数线）确实被称为“连续体”，但康托尔的连续体是通过单点定义的！亚里士多德和德谟克利特都会感到震惊！直到20世纪60年代，随着非标准分析的出现，连续体的思想才像凤凰一样从灰烬中再次崛起。我们将在本书的最后报告这一发展。

关于德谟克利特，据说[Edwards 1979]他发现了圆锥和四面体的体积公式：

V = 1/3 A · h

其中A表示底面积，h表示这些实体的高度。然而，这只是由欧多克索斯（Eudoxus）证明的。德谟克利特想象固体是由无限多个零厚度的薄片组成。今天我们称这些薄片为“不可分体”。然而需要注意的是，

德谟克利特当然没有关于不可分体理论的观念[Heath 1981, Vol. I, p. 181]。点、线和曲面分别是一维、二维和三维空间中的不可分体，因为它们的一个维度是零。在光谱的另一端，与原子论者的想法截然相反的是，连续体支持者认为固体是由连续体——因此也是固体——组成，而这些固体本身也是可以任意分割的。那些有限厚度的薄片今天被称为“无穷小量”。欧多克索斯在欧几里得《几何原本》第十二卷第5命题中证明德谟克利特体积公式的方法依赖于四面体的细分，但我们完全有理由相信，德谟克利特是在想象一个金字塔是由无限多个不可分体（平行于底面积的平面切割，如图2.2.3(a)所示）构成时得出这个结果的。巴尔特尔·范德瓦尔登（Baertel van der Waerden，1903–1996）在他的著作《科学觉醒》[van der Waerden 1971, p. 138]中引用了普鲁塔克的话（另见[Heath 1981, Vol. I, p. 179f.]），普鲁塔克将以下论点归因于德谟克利特：

“如果一个圆锥被平行于底面的平面切割，那么截面是相等还是不相等？如果它们不相等[我们可以在心中补充，如果薄片被视为圆柱体]3，那么圆锥将呈现阶梯状；但如果它们相等，那么所有截面都将相等，圆锥将看起来像一个由相等圆组成的圆柱体；但这完全是荒谬的。”

2.2.4 芝诺悖论

芝诺扮演了什么角色？请记住，他想捍卫他的老师巴门尼德，并想证明一切运动都只是幻觉。亚里士多德报告了芝诺的四个悖论，我们现在将开始讨论。

最广为人知的悖论是阿基里斯与乌龟，但二分法、飞箭和体育场通过亚里士多德的记载而流传至今。

阿基里斯与乌龟：

快跑运动员阿基里斯被要求与一只乌龟比赛。由于阿基里斯比乌龟快得多，乌龟获得相当大的领先。现在芝诺说：阿基里斯永远追不上乌龟！他推理如下：当阿基里斯到达乌龟的起点时，乌龟在他前面还有一小段距离。当阿基里斯到达这一点时，乌龟在他前面仍然有一（非常）小段距离，依此类推。为了说明这个论点，我们假设乌龟领先10米，阿基里斯比乌龟快10倍。阿基里斯跑10米只需要1秒。如果比赛开始，阿基里斯在1秒后将到达乌龟的起点位置。在这1秒内，乌龟领先阿基里斯1米。现在阿基里斯必须跑1米才能到达乌龟的位置（他只需要0.1秒），但那时乌龟仍然领先他10厘米。当阿基里斯跑完10厘米时，乌龟仍然领先他1厘米，依此类推，无限循环。

二分法：

芝诺说：一个人不能从A点移动到B点，A ≠ B。因为要从A到B，一个人必须先走完一半的距离。要走完一半的距离，一个人必须先走完四分之一的距离；要走完四分之一的距离，一个人必须先走完八分之一的距离，依此类推。因此，必须走完无限多个距离才能从A到B，这在有限时间内是不可能的。因此，运动是不可能的。

飞箭：

根据芝诺的说法，从弓中射出的箭不会飞。假设箭正在飞行，并在飞行中的某个点冻结时间。在这一时间点，箭的尖端位于空间的固定点，其速度为零（因为在这一时间点箭是固定的）。由于这一时间点可以在箭的飞行过程中任意选择，因此箭在任何地方的速度都为零。因此，箭根本不飞。

体育场：

想象你正在观看一场古代战车比赛，每辆战车有8人，参见图2.2.6。其中一辆战车由B队人员驾驶，另一辆由C队人员驾驶，同时另有8名A队人员固定在看台上观看。B战车向右移动，C战车向左移动，速度相同。当B战车行驶过一个A位置时，B和C战车已经行驶了两个位置！

根据芝诺的说法，这意味着已逝时间的一半等于已逝时间，这种矛盾再次暗示运动是完全不可能的。

现在可以立即反驳体育场悖论，因为芝诺显然没有意识到相对速度的概念，这个概念可以解决悖论。阿基里斯与乌龟的悖论对现代数学家来说也不是一个真正的问题，因为阿基里斯行进的总距离（以米为单位）是

10 + 1 + 0.1 + 0.01 + ... = 10 + ∑k=0∞ (1/10)k

这个无限级数是一个收敛的几何级数，其值为10/9。距离10 + 10/9 = 11.11111... 正好是阿基里斯追上乌龟的距离。但这并不是重点！首先，我们在这里使用了19世纪的知识，其次，这样的论证无法解释实际问题。为了说明，让我们关注阿基里斯与乌龟的悖论：芝诺提出的实际问题是关于空间（本例中是赛道结构）和时间结构的。如果我们将实数作为基础，悖论就可以解决，但谁能告诉我们，真实空间和真实时间实际上可以用实数建模呢？在1992年的一篇文章中，已故的约亨·霍普纳 [Höppner 1992, p. 59-69] 探讨了这个问题

如果我们现在从原子论和连续体理论的差异角度来看待芝诺悖论，那么我们就能识别出两组悖论：阿基里斯与乌龟和二分法旨在反对连续体的假设，并表明如果假设连续体（即任意可分性），就会出现严重问题。飞箭和体育场旨在反对原子论的假设，并表明这种假设也会导致严重问题。想象时间是由原子构成的，因此是一系列时间点，那么，正如芝诺所说，我们可以在这些时间点中的一个点上看到箭是静止的。这个观察结果似乎与箭的运动不符！另一方面，想象阿基里斯与乌龟悖论中的赛道是连续体，那么奔跑者将不得不经过连续体中无限多的部分，这些部分越来越小。芝诺说，这在有限时间内是无法完成的。

将这一切翻译成分析学的语言，我们正处于两种不同观点的萌芽阶段，这些观点甚至在今天仍有影响。伟大的莱布尼茨（Leibniz，1646-1716）最终将成为一位无穷小量数学家，因此他不会被无穷小量的计算所困扰。伟大的艾萨克·牛顿（Isaac Newton，1643-1727，按旧历为1642-1726）在物理学思想上倾向于原子论，甚至以原子论的方式思考光。

连续体和原子论的基本思想是如何进入西方文化圈的？这要归功于亚里士多德，他的译者，以及中世纪基督教经院哲学家对亚里士多德压倒性的兴趣。我们稍后将对此进行报告。

芝诺及其悖论至今仍备受争议。伟大的英国数学家兼哲学家伯特兰·罗素（Bertrand Russell，1872-1970）对芝诺的思想非常兴奋，甚至认为芝诺是19世纪数学的先驱，特别是卡尔·魏尔斯特拉斯[Russell 1903, p. 346ff.]。这当然是言过其实了。另一方面是巴尔特尔·范德瓦尔登，他试图边缘化芝诺的作用。他在[van der Waerden 1940, p. 141ff.]中辩称，原子论是在芝诺之后发展起来的，是对埃利亚学派的反击。此外，他声称毕达哥拉斯学派从未表现出对无穷小方法的真正兴趣。这个观点也走得太远了，走向了另一个极端。

可以肯定的是，关于直线结构的问题是分析的根源，所有研究人员，如牛顿、莱布尼茨及其继承者都受到了它的影响。然而，亚里士多德的连续体不能与我们最近的集合论关于实数线是“连续体”的观点相协调。我们还必须指出，亚里士多德著作中的连续体总是与运动问题紧密相关[Wieland 1965]。正是这种关于运动本质的思考将连续体的问题引入基督教经院哲学。

2.3 阿基米德

尽管古希腊的数学可能显得辉煌，但最耀眼的存在——一位全才——是阿基米德（Archimedes，约公元前287年-公元前212年）。他的活动范围是西西里岛上的锡拉库萨城，当时该城属于希腊领土。阿基米德很可能就出生在锡拉库萨。

2.3.1 生平、死亡与轶事

关于这位天才的生平，我们所知甚少，令人不安。然而，一些轶事流传至今，我们必须对其真实性保持非常谨慎。据说他发现杠杆定律时曾说：“给我一个立足点，我将移动地球”（引自亚历山大港的帕普斯）。更著名的是关于希耶罗二世国王（Hiero II，约公元前306年-公元前215年）王冠的故事。阿基米德当时在国王宫廷，很可能还是国王的亲戚。据说希耶罗二世下令制作第二个王冠，作为原版王冠的精确复制品，尽管两个王冠重量相同，但他怀疑金匠在复制品中掺假。阿基米德被指派调查此案。为了能放松思考，他去了一家澡堂，躺在温暖的水中。

2.3.2 阿基米德著作的命运

公元1204年4月，君士坦丁堡发生了前所未有的流血事件，并沦陷了。那些本想"解放"耶路撒冷的基督教十字军，侵占了欧洲最辉煌的城市；他们玷污了圣索菲亚大教堂，进行了掠夺、抢劫、强奸，并摧毁和移走了几个世纪以来在君士坦丁堡收集的图书。其中还有三本阿基米德的著作；即所谓的A、B和C抄本。

抄本A和B找到了通往西西里岛的路，但在1266年贝内文托战役后，它们被卖给了教皇[Dijksterhuis 1987, p. 37]。抄本B最后一次被提及是在1311年。此后抄本A消失了；1491年它在意大利人文主义者乔治·瓦拉手中。他去世后，抄本被卡普里亲王阿尔贝托·皮奥买下，然后于1550年传给了他的侄子红衣主教罗多尔福·皮奥，红衣主教于1564年去世。那一年之后，抄本A的踪迹就消失了。文艺复兴时期的大家从抄本A和B中汲取了阿基米德著作的知识。只有抄本C一直处于失落状态。仅凭抄本A和B中包含的著作，阿基米德就可以被认定为一位伟大的数学家和物理学家，但正是抄本C将阿基米德推上了不朽者的天堂，并使他与牛顿和莱布尼茨齐名。抄本C的历史是一个犯罪故事——不，是一个惊悚故事——连阿瑟·柯南·道尔也想不出更好的情节。这个故事在[Netz/Noel 2007]中有描述，我们将遵循其主要特点。1906年夏天，丹麦语文学家约翰·路德维希·海伯格（1854-1928）前往君士坦丁堡，检查梅托基翁（教会使馆教堂）里一份奇怪的手稿。此前他从1899年的一份目录中获得了关于一份羊皮纸重写本的信息，这立刻吸引了他。羊皮纸重写本是被重复使用的羊皮纸——鞣制的山羊皮——它们已经写过字。由于羊皮纸是昂贵的原材料，所以作者和抄写员重新使用旧的已写过的羊皮纸也就不足为奇了。他们刮掉了旧的铭文，将羊皮纸裁剪成新的格式，然后再次写上他们的著作。上述目录的作者，一位名叫帕帕多普洛斯-凯拉梅乌斯，没有固定的职位，而是根据他为目录撰写的页数获得报酬。因此，他提供了相当详尽的描述。他不仅描述了重写本上的新文本，还描述了原始羊皮纸上刮得不完美的、他仍然能够阅读的部分。语文学家

2.3.3 方法：关于机械定理的途径

根据海伯格的说法，阿基米德的《方法》被称为《阿基米德论机械问题的处理方法》，但我们已经指出，"方法"一词实际上并未出现在标题中。相反，阿基米德的这部作品应该更名为《途径》，而不是《方法》。因此，正确的标题应该是[Knobloch 2010]

Access with regard to mechanical theorems

（Zugang hinsichtlich der mechanischen Sätze）。

希思版本中的《途径》以[Heath 2002, p. 12, Appendix after p. 326]中的这些词开始：

'阿基米德向埃拉托斯尼问好。

我曾在前一次机会向你寄送了我发现的一些定理，只写出了陈述，并邀请你发现证明，而当时我并未给出证明。'

然后他开始告诉埃拉托斯尼，他将提供以下证明。一开始，他重复了一些关于重心定理的定理，这些定理他已经在他的著作《论平面的平衡或平面的重心》[Heath 2002, p. 189ff.]中给出了证明。在这部作品中，我们特别找到了杠杆定律，这是阿基米德以纯粹的公理方式推导出来的。整个杠杆理论仅基于三个公理[Heath 2002, p. 189]：

  1. 等重在等距处处于平衡，不等重在不等距处不处于平衡，但会向距离较大的那个重物倾斜。
  2. 如果当某些重物在一定距离处处于平衡时，向其中一个重物添加一些东西，它们将不处于平衡状态，但会向添加了东西的那个重物倾斜。
  3. 同样，如果从其中一个重物上取走任何东西，它们将不处于平衡状态，但会向没有取走东西的那个重物倾斜。

最终，阿基米德在命题6和7中证明了杠杆定律[Heath 2002, p. 192]。用我们的话说：

一个大重量G，距支点D处，与一个较小重量g在距支点d处平衡，如果

D : d = g : G (2.2)

成立。

借助这种机械方法，阿基米德现在继续称量不可分量！

MX / OX = KC / KN

抛物线段的面积是三角形ACZ面积的三分之一。

现在不难看出，内接在抛物线中的三角形ABC的面积仅是三角形ACZ面积的四分之一。这表明：

抛物线段的面积是三角形ABC面积的四分之三。

圆柱体体积 = 底面积 × 高度。

圆柱体的高度是AD，其底面积是π·BD²，其重心K正好位于AD的中间。我们的y轴现在指向右侧，因为我们将抛物体旋转了90°。从A到抛物体上的线段就是我们的y值。因此

BD² / OS² = AD / AS

因为AD就是当x值为BD = MS时，抛物线的y值。那么也应该有

MS² / OS² = AD / AS

因此

AS · MS² = AD · OS².

由于我们的前提是AD = AH，我们也可以写成

AS · MS² = AH · OS²。

但我们正在处理的是实体而不是面积，而且我们的不可分量不是线而是圆盘。因此实际上我们有

AS · (π· MS²) = AH · (π· OS²)，

AH · 抛物体体积 = AK · 圆柱体体积。

½AD · 抛物体体积 = ½AD · 圆柱体体积

因此：

旋转抛物体的体积正好是其包含圆柱体体积的一半。

2.3.4 通过穷竭法计算抛物线的面积

我们可以合理地假设阿基米德自己不接受称量不可分量的方法，参阅[Cuomo 2001]。这种方法过于惊人。因此，阿基米德的著作中只包含基于双重反证法的经典证明。在他的著作《抛物线的求积》中，他给出了抛物线段面积的另一个证明，这个证明与他在《方法》（途径）中给出的证明根本不同。他采用穷竭法进行证明，即用三角形填充抛物线段。

2.3.5 论螺旋线

在他的著作《论螺旋线》中，阿基米德考虑了现在以他命名的螺旋线。在极坐标形式下，它表示为

r = a/(2π) · φ, φ ≥ 0。

参数a > 0可以自由选择，φ是位置向量的角度，其长度r取决于φ。对于每个角度φ，我们因此找到螺旋线在位置向量尖端的相应点。阿基米德计算了螺旋线下的面积，我们想详细描述他解决这个求积问题的方法。

阿基米德考虑了螺旋线在完整旋转一周后的情况，即r = aφ/(2π)，φ∈[0, 2π]，以及如图2.3.14左侧所示的外接圆C。圆C的半径显然是a，因为螺旋线旋转一周后（φ=2π）位置向量的长度为r=a。我们用A表示半径为a的圆的面积，用B表示半径为b的任何其他圆的面积。我们从(2.1)中得知圆面积之比等于半径平方之比，即：

A / B = a² / b²

因此，对于其面积

Ak = π · l² = π · (k·a/n)²

所有扇形面积的和，对于k = 1, 2, ..., n，然后得出值

引理：

不等式

1² + 2² + ... + (n − 1)² < n³/3 < 1² + 2² + ... + (n − 1)² + n²，

成立。

当然，阿基米德严格证明了该引理。将此引理应用于(2.6)我们得到

(1² + 2² + ... + n²)/n³ A > n³/(3n³) A = 1/3 A。

我们现在考虑图2.3.15(b)所示的扇形。我们用Bk表示第k个扇形右边缘（因此朝向扇形k − 1）与螺旋线相交的圆的面积。角度φk−1显然是

φk−1 = (k − 1) 2π/n

因此，对于圆的半径，有：

L = (a/(2π)) (k − 1) 2π/n = (k − 1)·a/n

因此我们得到

Bk = π · L² = π · ((k − 1)·a/n)²。

由此得出

Bk / A = ((k − 1)·a/n)² / a² = (k − 1)² / n²

或

1/n Bk = ((k − 1)²/n³) A。

所有这些扇形面积从k = 1到k = n的总和因此是

(1² + 2² + ... + (n - 1)²)/n³ A。 (2.7)

我们现在应用引理并得到

(1² + 2² + ... + (n - 1)²)/n³ · A < n³/(3n³) · A = 1/3 A。

显然，面积(2.6)大于所求的螺旋面积F，并且同样显然，面积(2.7)小于F。因此它成立

(1² + 2² + ... + (n − 1)²)/n³ · A < F < (1² + 2² + ... + n²)/n³ · A。 (2.8)

我们还用我们的计算表明

(1² + 2² + ... + (n − 1)²)/n³ · A < 1/3 · A < (1² + 2² + ... + n²)/n³ · A (2.9)

成立。我们现在可以以两种不同的方式进行：首先，过高估计和过低估计面积之间的差是

(1² + 2² + ... + n²)/n³ · A - (1² + 2² + ... + (n − 1)²)/n³ · A = 1/n · A

n越大，差值越小。在极限情况下（阿基米德不知道这个概念）n → ∞时，差值将消失。因此，这个极限正是1/3 A，这可以从(2.9)得出，进而从(2.8)得出

F = 1/3 A。

其次，我们可以回顾计算平方和的公式，它可以在任何公式集中找到。它成立

1² + 2² + ... + n² = (2n³ + 3n² + n)/6

和

1² + 2² + ... + (n − 1)² = (2n³ - 3n² + n)/6

因此(2.9)可以写成

(2n³ - 3n² + n)/(6n³) · A < 1/3 · A < (2n³ + 3n² + n)/(6n³) · A，

在将分数除以n³后我们得到

(1/3 - 1/(2n) + 1/(6n²)) · A < 1/3 · A < (1/3 + 1/(2n) + 1/(6n²)) · A。

如果现在n无限增长，左侧和右侧都收敛到1/3 A，我们再次得到螺旋面积F正好等于圆A面积的三分之一的结果。

阿基米德不得不走一条更艰难的道路来完成最后一步，并采用了他的双重反证法。

作为精确值，他甚至"证明"了它。卡尔·西奥多·海塞尔住在俄亥俄州克利夫兰。在他的著作《看！大问题——圆的平方不再未解决》中，出版于1931年，他向世界展示了他最伟大的发现，即π的值为

256/81 ≈ 3.16049。

这个值正是艾赫迈斯在公元前1700年古埃及记载的值！

甚至在1960年，本丛书的编辑海因茨-威廉·阿尔滕，应联邦总统办公室的请求（！）检查了一位业余数学家的论文，他声称自己找到了"真实的数字π"（不用说，这是一个有理数）。这位业余人士是一位犹太移民，他声称自己的发现被纳粹以种族主义理由压制。因此，他的发现没有进入德国教科书。他声称自己不仅遭受了经济损失，还遭受了健康损害，因此他请求联邦总统提供10000德国马克5，以便在以色列开始治疗。

5 请注意，1960年英镑与德国马克汇率为1比11.8。

2.4 罗马人的贡献

我们可以非常简短地说明。罗马帝国的居民牢牢控制着地中海地区，修建了渡槽，征服了当时已知世界的大部分地区，他们对数学完全不感兴趣。然而，他们创造了罗马数字，中欧人一直使用到16世纪。这些数字至今仍被用作页码，并可在旧钟表和墓碑上看到。但我们只知道罗马人对分析学的一个贡献：

正如普鲁塔克在《马塞卢斯生平》[Plutarch 2004, p. 481]中所述，阿基米德曾指示他的朋友在他的墓前放置一个雕塑，雕塑展示一个球体和一个圆柱体，并刻有它们各自体积比例的公式。显然，他的愿望得到了实现。当伟大的马库斯·图利乌斯·西塞罗（公元前106-43年）于公元前75年在西西里担任财务官6时，他寻找阿基米德的坟墓，发现它状况不佳，于是下令挖掘并修复它。

西塞罗本人在[Cicero 1877, p. 186f.]中报道：

图 2.4.1. 西塞罗发现阿基米德之墓 ([本杰明·韦斯特 1797年画作] 耶鲁大学艺术画廊，纽黑文)
6 古罗马的公共官员；职业生涯中最低的等级，最终成为元老。

'我将向你们介绍一位同城谦逊而默默无闻的数学家，名叫阿基米德，他生活在许多年以后；他的坟墓被灌木和荆棘覆盖，我在担任财务官时发现了它，而叙拉古人对此一无所知，甚至否认还有任何遗迹；因为我记得一些诗句，我曾被告知这些诗句刻在他的纪念碑上，并且指出坟墓顶端放有一个球体和一个圆柱体。当我仔细检查了所有纪念碑（因为在阿克拉迪纳门外有许多坟墓）时，我看到一个柱子，略高于荆棘，上面刻有一个球体和一个圆柱体的图案；于是我立即对叙拉古人说——当时他们的一些主要人物和我在一起——我想那就是我正在寻找的。一些人被派去带着镰刀清理道路，为我们开辟了一条通道。当我们能够靠近它，走到基座前面时，我发现了碑文，尽管所有诗句的后半部分几乎都被磨损了。因此，希腊最崇高的城市之一，同时也是一度非常著名的学术中心，如果不是一位来自阿皮努姆的本地人发现了它，他们将一无所知他们最伟大天才的纪念碑。'

正如西塞罗在他的著作[Cicero 1931, p. 395]第五卷中所写，他还曾访问过梅塔蓬图的毕达哥拉斯的故居和逝世地：

'但是，皮索，我同意你的看法；地方强烈刺激想象力，并使我们对名人产生生动想法，这是普遍的经验。你还记得我曾和你一起去梅塔蓬图，而且在看到毕达哥拉斯辞世的那个地方以及他曾坐的座位之前，我都不愿去我们本该住的房子。'

老实说，这不是罗马对分析学的唯一贡献。罗马测量员——他们被称为Agrimensoris——不得不测量地貌中曲线形状的面积。他们通过"线性化"边界来完成任务。曲线边界被有限的割线分段取代，因此面积的一部分在线性化边界内部，另一部分在外部。这样一来，就可以得到可计算的面积部分。线性化的思想是积分微积分的基础，[Cantor 1875, p. 96], [Hinrichs 1992]。然而，所有这些初步工作都是由希腊人完成的，最近由莱尔格曼在[Lelgemann 2010]中进行了考察和发表。
